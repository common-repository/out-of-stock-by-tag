Out of stock by Tag, will allow you to bulk set products as out of stock by either Tag or Categories.

== Description ==

This plugin allows you to bulk set products as out-of-stock by either Tag or Category, via a very simple
menu in the backend of your wordpress installation. Tags and product categories available are being
automatically detected from which admins can select.

Please read plugin configuration details here: https://devwell.de/out-of-stock-by-tag/#configuration

For more detailed explanation look at https://devwell.de/out-of-stock-by-tag/

== Installation ==

1. Upload `out-of-stock-by-tag` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the tags and categories from the backend menu, woocommerce -> Out of stock

You can view plugin details here: Please read plugin configuration details here: https://devwell.de/out-of-stock-by-tag/#configuration
